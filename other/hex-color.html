2. HEX 与 RGB 颜色代码指南

2.1. 基础定义

类型 格式 示例
HEX #开头 + 6位十六进制 #FF5733
RGB 三个0-255的整数 rgb(255,87,51)

2.2. 转换规则

  1. HEX转RGB：
    • 拆分 HEX 为三组（如 FF 57 33）
    • 将每组十六进制转为十进制（FF=255，57=87，33=51）
  2. RGB转HEX：
    • 将每个十进制数转为两位十六进制（255→FF，87→57，51→33）
    • 组合成 #FF5733

2.3. 使用场景

  • ✓ 网页开发首选 HEX（CSS简写）
  • ✓ 动态调色使用 RGB（如 rgba(255,0,0,0.5) 透明效果）

3. RGBA 使用指南

3.1. 核心定义

参数 说明 取值范围
R(红) 红色通道强度 0-255
G(绿) 绿色通道强度 0-255
B(蓝) 蓝色通道强度 0-255
A(Alpha) 透明度（0=全透明，1=不透明） 0.0-1.0

3.2. 使用场景

  1. 半透明遮罩层：rgba(0,0,0,0.5) → 50%黑色遮罩
  2. 渐变背景：linear-gradient(rgba(255,0,0,0.3), rgba(0,0,255,0.7))
  3. 按钮悬浮效果：hover 时增加透明度（如从 1 → 0.8）

3.3. 代码示例

/* 半透明红色背景 */
.element {
  background: rgba(255, 0, 0, 0.5);
}

/* 带透明边框 */
.card {
  border: 2px solid rgba(0, 128, 0, 0.3);
}

/* 文字阴影效果 */
.text {
  text-shadow: 2px 2px rgba(0, 0, 0, 0.2);
}
