1. 进入全文本的linux桌面环境后要办的事情

1.1. 编辑~/.bash.rc or /etc/bash.bashrc 文件添加如下字符

就能在自己的用户登录的时候选择英文语系了, /etc/bash.bashrc是所有的用户都使用的文件 use english 

export LANG=en_US.UTF-8
export LANGUAGE=en_

1.2. 启动终端输入命令

sudo apt-get install locales

1.3. 重新配置LOCALE

dpkg-reconfigure locales
  1. 第一个页面将能勾选的项目全部勾选完成
    • 将能够勾选的全部勾选zhCN和enUS主要是
    • 勾选完成后会加载以下,完成后会提示
  2. 第二个页面选择默认的与语系环境
    • enUS.UTF-8或者是zhCN.UTF-8
  3. 运行LOCALE命令来查看当前的LOCALE环境
    • 成功后会提示的

1.4. 安装字体和输入法

设置完成必要的时候还是需要中文输入和显示的这时候怎么办呢 

apt-get install ttf-wqy-zenhei
apt-get install xfonts-intl-chinese wqy*
apt-get install ibus ibus-gtk ibus-pinyin
  • 安装之后就能够在ibu的设置里面添加任何你想要的输入法
  • 设置快捷键,测试.

2. locale指令

运行locale指令得到当前系统编码设置的详细资料

2.1. 指令详细解释

  1. 语言符号及其分类(LCCTYPE)
  2. 数字(LCNUMERIC)
  3. 比较和排序习惯(LCCOLLATE)
  4. 时间显示格式(LCTIME)
  5. 货币单位(LCMONETARY)
  6. 信息主要是提示信息,错误信息, 状态信息, 标题, 标签, 按钮和菜单等(LCMESSAGES)
  7. 姓名书写方式(LCNAME)
  8. 地址书写方式(LCADDRESS)
  9. 电话号码书写方式(LCTELEPHONE)
  10. 度量衡表达方式(LCMEASUREMENT)
  11. 默认纸张尺寸大小(LCPAPER)
  12. 对locale自身包含信息的概述(LCIDENTIFICATION)
